# Lenspy Proje Özeti

Lenspy, Python ve Tesseract tabanlı, minimal ayarlarla çalışan bir komut satırı OCR aracıdır. `input/` klasöründeki `.jpg`, `.jpeg` ve `.png` dosyalarını listeler, birini seçmenizi sağlar ve `output/<gorsel_adi>.txt` dosyasına işlenmiş metni kaydeder. README ve bellekte tanımlanan süreçler, bu akışın otomatik ve kullanıcı dostu kalmasını hedefler.
